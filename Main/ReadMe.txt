========================================================================
    控制台应用程序：MiniSql 类文件概述
========================================================================

Attribute.h
	定义了Attribute类，表示一个属性，可有三种类型（int, float, char(n)）
Condition.h
	定义了Condition类，表示判定条件，用于where子句。
File.h
	定义File类，表示文件，内有Block Head指针。
	定义Block类，表示块，相同文件的块会以双向链表串起来，
	而块中的offset值表示该块在当前文件的偏移量，
	如offset = 1，则该块数据是从文件的第BLOCKSIZE*offset+1位置开始读的.
	一个文件的文件头的块用于记录Table的元数据，包含：
		1. 使用大小
		2. Record数目
		3. 主键位置（属性计数从1开始）
		4. 属性数目
		5. 所有属性信息
	非文件头只包含1
BufferManager.h
	定义BufferManager类，具体函数说明见注释。
	另外未开放写入类，只要析构得当，所有缓冲区的数据会被写回硬盘。
CatalogManager.h
	定义CatalogManager类，处理元数据。
	Table元数据记录在各个Table文件，而Index数据单独记录在IndicesInfo文件中。
RecordManager.h
	定义RecordManager类，用于处理Record。
	在增删操作中，未对unique以及表元数据进行修改，因为涉及IndexManager的功能，
	请在API层加上这些功能。
